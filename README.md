# QtGMD-SecureRemote
基于 Qt + 物联网技术的三端远程设备控制系统，实现移动端/网关端指令下发、设备端协议转换与指令执行，适配工业物联网远程控制场景。

## 📋 项目简介
本项目采用**网关端-移动端-设备端**三端分离架构，围绕 MQTT 协议实现跨设备通信，结合 JSON 数据封装、Modbus 协议转换、密码+指纹双重验证等技术，完成远程设备的安全管控：
- 网关端：作为核心中转节点，提供密码+指纹登录验证，封装/解析 JSON 指令并通过 MQTT 转发；
- 移动端：Android 端可视化操作界面，实现 JSON 指令的发送与状态接收；
- 设备端：基于stm32 和 ESP8266 搭建 MQTT 环境，将 JSON 指令映射为 Modbus 协议，运用FreeRTOS通过消息队列处理设备控制逻辑。

## 🏗 项目架构
### 三端角色与交互逻辑
1. **移动端（MobileEnd）**：用户操作入口，编写 JSON 格式控制指令，通过 MQTT 发送至网关端；
2. **网关端（Gateway）**：
   - 安全验证：用户登录时完成密码+指纹双重校验；
   - 指令中转：接收移动端 JSON 指令，转发至设备端；接收设备端状态数据，回传至移动端；
3. **设备端（DeviceEnd）**：
   - 协议转换：接收 MQTT 传输的 JSON 指令，解析后映射为 Modbus 协议；
   - 指令执行：将 Modbus 数据送入消息队列，驱动底层设备完成操作，并反馈执行状态。
